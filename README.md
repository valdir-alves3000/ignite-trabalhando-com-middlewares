## Composição do nosso projeto

Neste projeto Temos alguns **Endpoints Base** que podem ser extendidos da forma mais adequada para seu contexto.

São eles:

### Usuários

- GET /users
- GET /users/:id
- PATH /users/:id/pro
- GET /todos
- POST /todos
- PUT /todos/:id
- PATH /todos/:id/done
- DELETE /todos/:id
